Trabalhando com tags para formulário

Elementos para formulário são voltados á interação com o usuário. Entrada de dados, seleção de opções, processamento para manipulação de dados e etc.


Formulários no HTML são somente uma INTERFACE.

Para que o formulário funcione, é necessário combiná-lo com alguma linguagem de programação.
